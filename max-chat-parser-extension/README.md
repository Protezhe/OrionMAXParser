# MAX Chat Parser (Яндекс.Браузер)

Расширение для Яндекс.Браузера (совместимо с Chromium API), которое собирает текст сообщений из открытого чата на `https://web.max.ru/*` и экспортирует в `JSON` или `TXT`.

## Установка

1. Откройте Яндекс.Браузер и перейдите на `browser://extensions`.
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное расширение**.
4. Выберите папку:

```text
max-chat-parser-extension
```

## Использование

1. Откройте чат, например:
   `https://web.max.ru/-71899898655408`
2. Нажмите иконку расширения `MAX Chat Parser`.
3. Нажмите **Собрать**.
4. Для сохранения нажмите **Экспорт JSON** или **Экспорт TXT**.
5. Если чат динамически обновляется, включите **Старт наблюдения**.

## Если не находит сообщения

Верстка чата может меняться. В popup есть блок **Селекторы**:

- `Корень чата`
- `Контейнер сообщения`
- `Текст сообщения`
- `Автор`
- `Время`

Измените CSS-селекторы, нажмите **Применить селекторы** и повторите сбор.

## Важно

- Расширение читает только страницу, открываемую вами в браузере.
- Используйте его только для данных, к которым у вас есть законный доступ.
